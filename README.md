# RedCalc

Calculadoras de infusão para profissionais e estudantes da saúde.

## Desenvolvimento local

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Verificação

```bash
npm test
```

Os valores de referência clínica ficam em `app/data/medications.ts` e devem ser validados antes do uso assistencial.
